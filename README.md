# py

This folder contains learning exercises and a few small projects.

## Structure

- `learning/` - small practice scripts
  - `learning/try/` - try/except exercises
  - `learning/oop/` - OOP exercises
  - `learning/versions/` - CSV/subscriber exercises (v1-v4)
- `projact/` - subscriber manager project
- `sql/` - SQLite practice scripts
- `flask_api_project/` - another project

## Running

- Subscriber manager:
  - `python projact/main.py`
  - Data lives in `projact/data/subs.json`
- SQL practice:
  - Run scripts from the `sql/` folder
